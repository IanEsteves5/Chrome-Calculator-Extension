Калкулатор за Google Chrome.
Проширење је 100% отвореног кода.

Карактеристике
 - Једноставни калкулатор са многим математичким функцијама
 - 2D графички калкулатор који може приказати више графикона истовремено
 - Без процеса у позадини
 - Страница за отклањање грешака која приказује стабло рашчлањавања и друге информације

Предлози су увек добродошли :)

Команде
 - help : приказује све команде
 - clear : чисти екран са резултатима
 - memory : приказује вредности сачуване у меморији
            употребљена меморија : <назив> := <вредност>
            пример : x := 3+10^2
 - log : приказује списак грешака за последње одрађено рачунање
 - functions : приказује доступне функције
 - plot : отвара графички калкулатор
 - tree : отвара прозор за отклањање грешака
